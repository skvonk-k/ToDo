# Список дел

Простое приложение для управления списком дел на WPF (C#, .NET Framework 4.8). Учебный проект для практики работы с XAML, биндингами и сохранением данных в JSON.

## Возможности

- Добавление, редактирование и удаление задач
- Отметка задач как выполненных (чекбокс в колонке "Статус")
- Автоматическая дата создания задачи
- Сохранение и загрузка списка дел в JSON-файл
- Визуальное выделение выполненных задач (прозрачность)

## Технологии

- C# / WPF
- .NET Framework 4.8
- Newtonsoft.Json — для сериализации/десериализации данных

## Как запустить

1. Склонируй репозиторий:
   ```
   git clone https://github.com/твой-username/название-репозитория.git
   ```
2. Открой `ToDo.sln` в Visual Studio
3. Восстанови NuGet-пакеты (обычно происходит автоматически при открытии)
4. Собери и запусти проект (F5)

## Структура проекта

```
ToDo/
├── Models/          # Модели данных (TodoModel)
├── Services/         # Сервисы (работа с файлами, JSON)
├── Icon/             # Иконка приложения
├── MainWindow.xaml    # Главное окно
└── App.xaml
```
